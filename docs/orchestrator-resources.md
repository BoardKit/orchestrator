## Lifecycle Summary

### 1. Clone Orchestrator
- All pre-configured resources ready
- Setup system present

### 2. Run Setup Wizard
- Generates SETUP_CONFIG.json
- Customizes CLAUDE.md and README.md
- Creates repo-specific skills, guidelines, settings
- Creates CLAUDE.md for each application repo

### 3. Create Symlinks
- Links application repos to shared resources

### 4. Delete Setup
- Removes entire `setup/` directory
- Keeps only operational files

### 5. Normal Operation
- Update resources in orchestrator
- Changes propagate via symlinks
- Use cross-repo-doc-sync to keep docs current
